# SQL Data Analytics Project

## 📌 Overview

This project demonstrates SQL data analysis techniques using a sales dataset. It focuses on answering business questions and generating meaningful insights through SQL queries.

## 🎯 Objectives

* Analyze sales and customer performance
* Identify top-performing products
* Track sales trends over time
* Segment customers and products
* Create business-ready reports

## 🛠️ Tools & Technologies

* SQL Server
* T-SQL
* Window Functions
* Common Table Expressions (CTEs)
* Aggregate Functions
